# term-paper
Примоченко И.В.  УИБО-14-24 "Калькулятор эффективности мероприятий по охране труда" на языке C++:
## Описание

Данная программа позволяет оценить эффективность мероприятий по охране труда путем расчета:
- Годовой экономии от снижения травматизма
- Срока окупаемости мероприятия
- Коэффициента эффективности

Программа включает:
- Ввод исходных данных
- Расчет показателей
- Автоматическое тестирование
- Интерпретацию результатов

## Требования

- Компилятор C++ (поддерживающий стандарт C++11 или выше)
- Операционная система: Windows, Linux, macOS

### Сборка и запуск

#### Linux/macOS:
```bash

./build_and_run.sh
